# sr-engineers discussion

スクリーン・リーダーを使っているエンジニアによる情報交換を目的に作成した github discussion のスペースです。

## 使い方

GitHub のアカウントを持っていれば、だれでも自由に投稿することができます。細かい使い方については、これから readme を整備していきます。

### 新しい話題を投稿する場合

とりあえず、今のところはまだトピックの整備ができていないので、

- 質問形は Q&A
- その他雑多なトピックは general

に投稿していただければ大丈夫です。
